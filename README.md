# React To-Do List Application

A simple React To-Do List application built using Vite.

## Features

- Add Tasks
- Edit Tasks
- Delete Tasks
- Mark Tasks as Completed
- Responsive UI

## Technologies Used

- React
- Vite
- CSS

## Installation

1. Clone the repository

```bash
git clone https://github.com/Saad-Shams-dev/react-to-do-app.git
```

2. Open project folder

```bash
cd react-to-do-app
```

3. Install dependencies

```bash
npm install
```

4. Run the project

```bash
npm run dev
```
